# Simon Says Game:
A fun and interactive memory-based game inspired by the classic "Simon Says" game. Test your memory by repeating the sequence of colors correctly!

## Features:
- Random color sequence generation
- Increasing difficulty level
- User input detection
- Game over alert on wrong input
- Simple and clean UI

## Tech Stack:
- HTML
- CSS
- JavaScript

## How to Play:
1. Press any key to start the game
2. Watch the sequence of colors carefully
3. Repeat the sequence by clicking the buttons
4. Each level adds a new color to the sequence
5. If you click the wrong color, the game ends
