## 📌 About

Simon Says is a classic memory-based game where the computer shows a random color sequence and the player has to repeat it.  
The sequence gets longer every round — how far can you go? 🎯

---

## ✨ Features

- 🎨 **Random color sequence** generation every round
- 📈 **Increasing difficulty** — sequence grows each level
- 🖱️ **User input detection** — click to match the pattern
- ❌ **Game Over alert** on wrong input
- 🔄 **Restart** anytime
- 🧼 Simple and clean UI

## 🎮 How to Play

1. Press any key or click **Start** to begin
2. Watch which color flashes
3. Click the same color to repeat
4. Each round adds one more color to the sequence
5. Wrong click = **Game Over!**
